# Git Deneme Projesi

Bu proje Git workflow ve branch yönetimi denemesi için oluşturulmuştur.

## Özellikler

- Master branch: Production-ready kod
- Develop branch: Development aşamasındaki yeni özellikler

## Kurulum

```bash
npm install
```

## Kullanım

```bash
node app.js
```

## Katkıda Bulunma

1. Develop branch'ine geçin
2. Yeni bir feature branch oluşturun
3. Değişikliklerinizi yapın
4. Pull request oluşturun
